# Pillars

## 1. Joyful Movement

Movement should feel good enough to be fun without objectives.

Players should enjoy:
- Running
- Jumping
- Shell dashing
- Slingshot traversal

Design questions:
- Does this make movement more fun?
- Does this interrupt movement?

Good fits:
- Bounce chains
- Shell Dash
- Slingshot traversal

Avoid:
- Heavy inventory management
- Systems that stop the player too often

## 2. Memorable Worlds

Every world should feel distinct and easy to describe.

Players should say:
- "I loved the candy world."

Not:
- "I forgot level 4."

Design questions:
- Does this world have a strong identity?
- Would a player remember this area?

Good fits:
- Lobstrocities
- Mechanical wolves
- Toybox world identities

Avoid:
- Generic grassland
- Worlds that only differ by color palette

## 3. Reward Curiosity

Players who explore should find something worth the detour.

Rewards can include:
- Grunk
- Cosmetics
- NPC dialogue
- Secrets
- Bobbleheads
- Weird little interactions

Design questions:
- Is there a reason to wander?
- Is this optional reward interesting?

Avoid:
- Mandatory fetch quests pretending to be exploration

## 4. Optional Weirdness

The game should make complete sense at surface level. Deeper ideas exist only for players who look.

Design questions:
- Can a kid enjoy this?
- Can an adult overthink this?

Good fits:
- Time is funny here
- Strange NPC memory
- Characters understanding things the player does not

Avoid:
- Required lore homework
- Explaining the magic out of the room

## 5. Expression Over Power

Players do not become stronger. Players become more themselves.

Progression should reward:
- Cosmetics
- Identity
- Completion
- Hub life
- Understanding

Design questions:
- Does this create self-expression?
- Does this invalidate earlier content?

Avoid:
- Damage upgrades
- Level-based power
- "+15% shell damage"
