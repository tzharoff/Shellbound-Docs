# Progression

## Philosophy

Shellbound progression changes appearance, expression, hub life, completion, and understanding. It does not increase stat power.

The player should feel:
- More connected to the world
- More expressive
- More aware of what is really going on
- More invested in the hub

The player should not feel:
- Numerically stronger
- Required to grind for combat stats
- Like earlier content has been invalidated

## Collectibles

Possible collectible categories:
- Junk
- Bobbleheads
- Cosmetics
- World-specific oddities
- Optional lore fragments
- Hub decorations

## Completion Tracking

Completion tracking should help curious players without making the game feel like homework.

Possible tracking:
- Beam restored
- Boss resolved
- Junk count
- Bobbleheads
- NPC interactions
- Secret discoveries

## Unlockables

Unlockables should focus on expression and discovery.

Examples:
- Shell skins
- Dash effects
- Slingshot variants
- Hub decorations
- Dialogue reactions
- Cosmetic shop inventory

## Endgame

Endgame is not defined yet.

Possible endgame focus:
- Final Great Turtle encounter
- Completed beam network
- Optional lore clarity
- Hub fully alive
- Secret shell-under-shell discoveries

## Replayability

Replayability should come from movement mastery, collecting, optional weirdness, and world familiarity.
