# Vertical Slice

## Scope

The vertical slice should prove the core Shellbound promise:
- Movement feels joyful.
- A small world is memorable.
- Junk collection works.
- Grunk conversion works.
- Shellvis sells expressive rewards.
- A beam can be restored.
- The hub changes afterward.
- Optional weirdness exists for curious players.

## Candidate Slice

Sunny Shore is the current strongest candidate because it can introduce:
- Cozy beach adventure tone
- Great Red Lobstrocity
- Junk collection
- First restored beam
- Simple movement routes
- A memorable boss with strange speech

See [Sunny Shore](sunny-shore.md) for the first-world flow.

## Success Metrics

The slice succeeds if players say:
- "Movement feels good."
- "I wanted to explore."
- "I understood what I was doing."
- "The world had personality."
- "That was weird, but in a good way."

## Required Features

- Third-person movement
- Camera
- Jump
- Shell Dash
- Basic slingshot interaction
- Junk pickups
- Grunk conversion
- Shellvis shop purchase
- One cosmetic or dash effect
- One beam restoration
- One small hub change
- One boss or boss-like encounter
- Dialogue overlay

## Stretch Goals

- Bobblehead collectible
- Shellbert secret interaction
- The Kid appearance
- Unique NPC reaction to cosmetic
- Emperor's New Shell test
- Optional lore breadcrumb
