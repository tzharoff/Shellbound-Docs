# Audio

## Music Direction

Music should support cozy retro adventure first, then let the strange edges leak in.

Possible qualities:
- Melodic
- Playful
- Low-poly nostalgia without direct imitation
- World-specific identities
- Subtle unease in ancient or time-weird spaces

## World Themes

Each world should have a memorable musical identity.

Needs:
- Sunny Shore theme direction
- Endless Tracks theme direction
- Beam Forest theme direction
- Forgotten Aquarium theme direction
- Sugarwind Highlands theme direction
- Skyward Shell theme direction
- Turtle Temple theme direction
- Turtles All The Way Down theme direction

## Sound Effects

Priority sounds:
- Footsteps
- Jump
- Shell Dash
- Slingshot
- Junk pickup
- Grunk conversion
- Cosmetic purchase
- Beam restoration
- Dialogue advance

## Character Voices

Voice style is not defined.

Possible direction:
