# Gameplay

## Moment-To-Moment Mix

Target gameplay breakdown:
- 40% traversal
- 20% exploration
- 15% combat
- 10% collecting
- 10% NPC interaction
- 5% weird discoveries

## Short Session Flow

A short session should support:
- Collect Grunk.
- Buy a cosmetic.
- Leave satisfied.

## Long Session Flow

A long session should support:
- Complete a world.
- Face a boss.
- Restore a beam.
- Return to a changed hub.

## Progression Philosophy

Progression is about expression, completion, hub life, and understanding.

The player should not out-stat earlier content. A beach crab should not become irrelevant because the turtle found math pants.

## Completion Philosophy

Completion should reward curiosity without making the game feel like a spreadsheet.

Possible completion vectors:
- World collectibles
- Restored beams
- Bobbleheads
- Cosmetics
- Hub decorations
- Optional dialogue discoveries
- Weird lore threads
