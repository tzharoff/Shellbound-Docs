# Movement

Movement is the first prototype priority.

## Baseline Moves

- Run
- Jump
- Camera control
- Shell Dash

## Goals

- Responsive third-person movement.
- Clear momentum without feeling slippery.
- Fun traversal in empty test spaces.
- Camera that supports exploration and platforming.

## Open Questions

- Does Shell Dash damage enemies, break objects, or only help traversal?
- Can the player chain movement actions?
- How expressive should aerial control be?
- Is the slingshot part of movement, interaction, or both?
