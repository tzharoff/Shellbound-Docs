# Shellbound

Shellbound is a nostalgic PS1-inspired third-person adventure collectathon about a small turtle restoring ancient beams, collecting junk, and wandering through strange connected worlds.

## Purpose

This repository holds the lightweight working docs for Shellbound: design notes, devlogs, production planning, reference material, and decision records.

Current status: **early concept / pre-prototype**

No docs generator, app framework, or build tooling lives here yet. Just markdown, vibes, and useful decisions.

## Folders

- `devlog/` - Session notes and progress updates.
- `design/` - Game vision, systems, worlds, characters, and core experience notes.
- `production/` - Roadmap, milestones, backlog, and risk tracking.
- `decisions/` - Design decision records for choices that should not evaporate later.
- `reference/` - Shared language, naming notes, and cut or parked ideas.
